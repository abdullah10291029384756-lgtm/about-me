# 🛒 مشروع سلة مشتريات (Cart)

## 💡 الفكرة
هذا المشروع يهدف إلى إنشاء واجهة بسيطة لسلة مشتريات باستخدام:
- **HTML** لبناء هيكل الصفحة
- **CSS** لتنسيق وتصميم واجهة المستخدم
- **JavaScript** لإضافة التفاعلات مثل إضافة/حذف المنتجات وحساب الإجمالي

## ⚙️ طريقة التشغيل
1. قم بتحميل المشروع أو نسخه من GitHub.
2. افتح الملف **index.html** في متصفح حديث.
3. تأكد أن الملفات (**main.js** و **style.css**) موجودة في نفس المجلد.

## 🧰 المزايا الحالية
- عرض المنتجات في الصفحة.
- إمكانية إضافة منتجات إلى السلة.
- إمكانية حذف منتجات من السلة.
- حساب المجموع الكلي للسلة.

## 🚀 الخطوات القادمة
- إضافة زر "إتمام الشراء".
- تحسين واجهة المستخدم بتصميم أكثر جاذبية.
- حفظ حالة السلة باستخدام Local Storage.
- ربط السلة بقاعدة بيانات أو API مستقبلاً.
